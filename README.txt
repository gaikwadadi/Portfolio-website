Aditya Gaikwad Portfolio
This portfolio website is built using the Massively template by HTML5 UP — a clean, article-oriented design featuring a large background image and smooth scroll effects powered by Scrollex. It has been customized to showcase my work, skills, and projects as a Data Scientist and Machine Learning enthusiast.

About the Design
The layout uses a visually striking parallax background with subtle scroll animations to enhance user engagement.

The site focuses on content clarity and readability, ideal for presenting detailed professional information.

Originally developed by HTML5 UP, the design is free to use under the Creative Commons Attribution 3.0 license.

Technology & Tools Used
HTML5 & CSS3 for semantic and responsive markup.

jQuery

Scrollex for scroll-driven animations.

Font Awesome icons to enrich UI elements.

Custom styling to match my personal branding and portfolio needs.
