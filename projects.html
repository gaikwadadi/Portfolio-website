Worked Projects

AI-Powered Quiz Generation Model

The Quiz Generator App is an interactive Streamlit web application that leverages Google Gemini LLM to create quizzes in multiple formats including multiple-choice, true/false, and open-ended questions. It uses structured JSON-based prompt engineering for reliable quiz generation, integrates fuzzy matching and normalization to validate user responses, and provides real-time scoring with feedback. The app features a clean UI for attempting quizzes, allows users to export quizzes as text files, and includes secure API key handling with retry mechanisms for robustness. This project demonstrates practical skills in LLM integration, prompt design, and building user-friendly AI-powered applications.

TRAFFIC LIGHT DETECTION USING YOLO V8

Built a deep learning model to detect traffic lights in diverse real-world conditions using the Small Traffic Light Dataset (S2TLD). Converted annotations from Pascal VOC to YOLO format, applied extensive data augmentation, and trained a YOLOv8 model with hyperparameter tuning. Evaluated performance using mAP, precision, and recall to ensure robust detection across varied lighting, weather, and angles. Implemented with Python, PyTorch, YOLOv8, OpenCV, and Albumentations, this project demonstrates practical expertise in object detection pipelines from dataset preparation to high-performance model deployment.

SPOTIFY MUSIC DATA ANALYSIS

Conducted an end-to-end SQL project analyzing a Spotify dataset containing track, album, and artist details along with audio features such as danceability, energy, loudness, and tempo. The process involved normalizing a denormalized dataset, executing SQL queries across easy, medium, and advanced difficulty levels, and applying performance optimization techniques like indexing and query plan analysis. The goal was to practice advanced SQL skills while extracting actionable insights from complex music data.
